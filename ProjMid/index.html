What is IPv6?

In order to connect devices over the Internet, each device must have an Internet protocol (IP) address. The current IP system is Version 4 (IPv4), which makes available over four billion IP addresses. However, the huge increase in Internet users and devices worldwide means that IPv4 addresses are running out. IPv6, the next-generation protocol, provides approximately 340 undecillion IP addresses, ensuring availability of new IP addresses far into the future, as well as promoting the continued expansion and innovation of Internet technology.
